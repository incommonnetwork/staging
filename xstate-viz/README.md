# xstate-viz

See it in action at https://statecharts.github.io/xstate-viz/.


## Super quick start

In the root and public directory (`xstate-viz/` & `xstate-viz/public`) run
```bash
npm i
```

In root run
```bash
npm link
```

In public run 
```bash
npm link @statecharts/xstate-viz
```

In root run
```bash
npm run develop
```
